HOW THIS WORKS
Two parallel funnels: 🏟 Wholesale = venues (breweries, event spaces, summer venues). 🏢 Catering = corporate offices. Same agent architecture, different tables & crons.
Lifecycle: scouted → qualified → ready → contacted → replied → active. With detours: hold · drew_flag · archive · unsubscribed.
Schedule (MT):
Mon 6am — wholesale scout (Apollo)
Mon 7:30am — catering scout
Mon/Thu 7am — qualifier
Mon–Fri 8am — wholesale outreach (~8 sends/run · target 32/wk)
Mon/Wed/Fri 8am — catering outreach (~7 sends/run · target 21/wk)
Every 15min — reply scanner (Gmail)
Approval gate: wholesale = first 10 lifetime sends. Catering = first 50 per rolling 30 days. Before the gate, drafts park for your approval. After, they auto-send.
When you must act: approve parked drafts · handle replies the agent can't · review drew_flag · fill missing contacts. All surfaced in the Inbox/Today list.
Drive it harder anytime: add leads, run any agent on demand, preview drafts, queue sends early, skip/edit individual follow-ups.
